Mafia Online V5

طريقة الميزاجور:
1) فك الضغط على هذا الملف.
2) ارفع إلى GitHub نفس المشروع فقط هذه الملفات:
   - index.html
   - app.js
   - style.css
3) لا ترفع firebase-config.js ولا تبدله، لأنه فيه إعدادات Firebase ديالك.
4) اضغط Commit changes.
5) انتظر دقيقة أو دقيقتين وافتح نفس رابط GitHub Pages.
6) إذا بقيت النسخة القديمة، افتح الرابط في navigation privée أو امسح cache.

الجديد في V5:
- روابط Instagram لرشيد بابا وعبد الصمد بابا في أسفل الصفحة.
- ظهور دور اللاعب بعد إقصائه.
- لائحة المقصيين تحت الأحداث.
- إحصائيات: عدد المافيا المتبقين، الفريق الصالح، الأحياء، والمقصيين.
- اللائحة العادية للاعبين ولات تحت الأحداث.
- منع إنشاء غرفة بنفس الرقم إذا كانت موجودة، مع رسالة تطلب الضغط على دخول.
- عدد المافيا: مافيا واحدة لكل 5 لاعبين تقريباً، يعني 10 لاعبين = 2 مافيا.
- المافيا يشوفون أسماء المافيا الآخرين في الدور السري ولوحة المافيا.
