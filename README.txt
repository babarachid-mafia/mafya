MAFIA FIREBASE TEST

هذه نسخة تجربة Online بلا PC وبلا Node.js.
تحتاج فقط Firebase Realtime Database + GitHub Pages.

الملفات:
- index.html
- style.css
- app.js
- firebase-config.js

خطوات عامة:
1) صاوب مشروع Firebase.
2) فعل Realtime Database في Test mode.
3) نسخ firebaseConfig من Web app.
4) لصق config في firebase-config.js.
5) ارفع الملفات إلى GitHub.
6) فعل GitHub Pages من Settings > Pages.
7) افتح الرابط وجرب الغرفة.

ملاحظة أمنية:
Test mode غير مناسب لمدة طويلة. استعمله فقط للتجربة.
